# GNU/Linux
## Descarga de repositorio
```$ git clone https://github.com/specter327/openshell-public.git```

## Apertura de repositorio
```$ cd openshell-public```

## Instalacion de requisitos
```$ bash scripts/install.sh```

## Consola
### Integracion de consola
```$ console --integrate```

### Provision de identidad
Compartir al administrador de sistema:
- Identificacion logica: UID
- Identidad criptografica: PIK

*La informacion debe estar completa e integra. Cualquier confusion, puede provocar un error durante la integracion.*

### Provision de pasaporte
El administrador de sistema debe entregar:
- Clave de seguridad del pasaporte de integracion provisto

De esta manera, la **CONSOLA** pasa a encontrarse en un estado OPERATIVO

## Agente
### Integracion de agente
```$ agent --integrate <OPEN_SECURITY_CODE>```

De esta manera, el **AGENTE** pasa a encontrarse en un estado OPERATIVO
